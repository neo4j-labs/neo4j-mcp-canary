## v0.2.3 - 2026-04-20
### Added
* mixPanel event returns estimated and actual rows to help determine default limit

* Cypher operations examine response from EXPLAIN to compare estimated rows against a limit.  if exceed, bails out
* Updated tests to reflect changes
